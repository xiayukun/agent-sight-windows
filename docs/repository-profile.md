## README 首屏卖点

- AI 通过真实像素观察 Windows GUI。
- 工具只提供像素、坐标、时间、证据路径，不做 OCR 或业务判断。
- 公共链路短：`screen -> look -> do -> look`。
- visual memory / attention toolbox 降低截图成本。
- Tray GUI 给人类可见暂停、允许、紧急停止和语言切换。
- Session Supervisor 是当前用户态驻留入口。
